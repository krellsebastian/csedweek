Installation von PocketCode

Es gibt zwei Möglichkeiten, wie man PocketCode installieren kann:

  • Vom PC aus
  • Vom Handy aus

Schritt 1: Wenn Sie gerade vor einem PC sitzen, klicken Sie auf folgenden Link: ab zur App.

Schritt 2: Klicken Sie nun auf 'Installieren'. Falls Sie nicht angemeldet sind, melden Sie sich an.

Schritt 3: Das App installiert sich jetzt automatisch auf Ihrem Handy. (Hinweis: Damit sich das App installieren kann, muss das Handy eine Internetverbindung haben)

Schritt 1: Öffnen Sie das Playstore-App.

Schritt 2: Geben Sie in die Suchleiste 'PocketCode' ein und wählen es aus. Klicken Sie anschließend auf 'Installieren'.

Schritt 3: Akzeptieren Sie die App-Bedinungen. PocketCode wird nun installiert.
